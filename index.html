Reference Table

Selector Name Description
  Descendant Selector Combines two selectors where elements matching the second selector are selected if they have an ancestor matching the firt selector
+ Adjacent Selector Placed betwen two selectors, matches only elements matched by second selector if they are the next sibling element of the first
> Direct Child Placed betwen two selectors, matches only elements matched by second selector if they are direct children of the first

Descendant Selector

Select all <a>'s that are nested inside an <li>

CSS

li a {
    color: teal;
}

HTML

<li>
    ...
    <a>

Adjacent Selector

Select only the paragraphs that are immediately preceded by an <h1>

CSS

hi + p {
    color: red;
}

HTML

<h1> 
<p>

Direct Child

Select only the <li>'s that are direct children of a <div> element

CSS

div > li {
    color: white;
}

HTML

<div>
    <li>
